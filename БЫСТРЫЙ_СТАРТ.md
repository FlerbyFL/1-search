## 🐛 ЕСЛИ ЧТО-НИБУДЬ НЕ РАБОТАЕТ:

### Backend не отвечает?
Перезапустите Go:
```powershell
cd C:\Users\Admin\Desktop\1-search\backend
go run .
```

### Frontend не загружается?
Перезапустите npm:
```powershell
cd C:\Users\Admin\Desktop\1-search
npm run dev
```

### Нет результатов в поиске?
Обновите БД:
```powershell
Invoke-WebRequest "http://localhost:8080/api/parse-all" -Headers @{"X-Parse-Token"="secret-parse-token-2026"}
```
