### Вариант 1: Через браузер (Рекомендуется)
1. Откройте: **http://localhost:3000** в браузере
2. Введите название товара (например: "смартфон", "ноутбук", "телевизор")
3. Смотрите цены из всех магазинов
4. Сравнивайте цены и выбирайте лучший вариант

### Вариант 2: Через API
```bash
# Поиск товаров
curl "http://localhost:8080/api/search?q=смартфон"

# Получить статистику
curl "http://localhost:8080/api/stats"

# Обновить БД (требует токен)
curl -H "X-Parse-Token: secret-parse-token-2026" \
     "http://localhost:8080/api/parse-all"
```

## 📞 ПОДДЕРЖКА

### Если возникли проблемы:

**Проблема**: Frontend не загружается
- Решение: Проверьте что npm run dev работает (в терминале)

**Проблема**: Поиск возвращает 0 результатов
- Решение: Запустите парсер: `Invoke-WebRequest "http://localhost:8080/api/parse-all"`

**Проблема**: Backend не отвечает
- Решение: Перезапустите: `go run .` в папке backend/

**Проблема**: Ошибка подключения к БД
- Решение: Проверьте что PostgreSQL запущена: `pg_isready`
