# 📁 Полный список файлов проекта

## Backend Go код

### Основные файлы приложения

1. **parser.go** (782 строк)
   - Основной парсер с HTTP обработчиками
   - Парсинг всех 6 магазинов параллельно
   - Endpoints: /api/search, /api/parse-all, /api/stats, /healthz
   - Особенности: Wildberries API, JSON-LD extraction, OpenGraph fallback

2. **database.go** (136 строк)
   - Инициализация подключения к PostgreSQL
   - Создание таблиц и индексов
   - Настройка пула соединений
   - Инициализация статусов парсинга магазинов

3. **repository.go** (270 строк)
   - Слой доступа к данным (DAL)
   - 11 функций для работы с БД
   - SaveProducts() - батч вставка
   - Поиск: по названию, магазину, цене
   - Управление: удаление старых, очистка магазина
   - Получение статистики

4. **utils.go** (12 строк)
   - defaultContext() - контекст 30 сек
   - longContext() - контекст 60 сек

5. **maintenance.go** (156 строк)
   - Обслуживание БД
   - PrintDatabaseStats() - вывод статистики
   - OptimizeDatabase() - VACUUM ANALYZE
   - BackupDatabase() - резервная копия
   - CleanupOldData() - удаление старых товаров
   - ExportProductsToJSON() - экспорт в JSON

### Конфигурация

6. **go.mod**

9. **Dockerfile**

## 🚀 Готовые команды

```bash
# Windows
cd backend && setup.bat

# Linux/Mac
cd backend && chmod +x setup.sh && ./setup.sh

# Docker
docker-compose up -d

# Проверка
curl http://localhost:8080/healthz
curl "http://localhost:8080/api/search?q=смартфон"
curl http://localhost:8080/api/stats
```
